## 🛠️ 사용 기술 및 라이브러리

- Swift, iOS, UIKit, CoreData

## 📱 기능 구현

- Clean - Architecture 적용
- CodeBase UI 구현
- Image 색상값 추출해서 배경 색상 변경
- CoreData를 활용해서 최근검색어 구현
- 무한 스크롤 컬렉션 뷰 구현
